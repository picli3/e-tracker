# e-track

E-paper based GPS tracker PCB design using KiCad.

## Project Structure

- `e-track.kicad_pcb` - Main PCB layout
- `e-track.kicad_sch` - Schematic file
- `e-track.kicad_pro` - KiCad project file
- `e-track.kicad_prl` - KiCad layout settings
- `Power_Manager.kicad_sch` - Power management module
- `e-paper.kicad_sch` - E-paper display module
- `docs/` - Datasheets and documentation

## Features

- E-paper display interface
- GPS module integration
- Power management system
